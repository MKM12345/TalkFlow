# TalkFlow

**TalkFlow** is a modern, lightweight messaging platform where users can **create and join chat rooms** using unique codes. It's designed for real-time group communication and currently in active development with more features coming soon.

## Features (Planned)

- Create a room and get a unique code
- Join a room with a code
- View the rooms you're currently in
- Real-time messaging for all members
- Clean and simple interface
- Future: user accounts, invites, moderation, and AI-powered features (undecided, but hopefully!)

## Tech Stack

- **Backend:** Node.js, Express, Socket.io
- **Frontend:** HTML, CSS, JS (can later upgrade to React/Next.js)
- **Database:** MongoDB or PostgreSQL
- **Still deciding:** Python microservice for moderation or AI features

Stay tuned! :)
